YITH Product Size Charts for WooCommerce allows you to create custom Size Charts.


== Description ==

= Features: =

* Possibility to create unlimited custom size charts
* Possibility to assign the Size Chart created to a product
* Size Chart displayed in a custom tab on single product page

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Product Size Charts for WooCommerce` from Plugins page

YITH Product Size Charts for WooCommerce will add a new submenu called "Product Size Charts" under "YIT Plugins" menu. Here you are able to view the features of premium version.
